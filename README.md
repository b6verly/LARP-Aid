*Second release since being totally public!* **V1.6.0** marks a pretty significant update. The Law Enforcement side of Larp-Aid has been overhauled, along with many other updates.

To download, navigate to the releases tab, and download the latest version. You will download the LarpAID-dist.zip file, then unzip it using winrar, 7zip, or another application. Once you have done those steps, the only thing that must be done is to run LARP-AID.exe . Do note, if you would like to pin Larp-AID to your taskbar/desktop, you must click on the .EXE file, and drag and drop it into your taskbar. If adding to desktop, please create a shortcut first via the right click menu. In future versions; **V1.6.0+** will have a shortcut included where you can move to desktop or taskbar easier. I will also make installation easier soon.
